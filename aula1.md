## Tipos de dados
### Aula 1 Estrutura de Dados

Int: 4 bytes
- Faixa: -2,147,483,648 a 2,247,648
- Aplicação: Contadores, indices de vetor, armazenamento de quantidade discreta

Short: 2 bytes
  - Faixa:
  - Aplicação: Economizar memória

Long e long long 4 ou 8 bytes
  - Aplicação: Valores muito grandes.

Float 4 bytes
- Aplicação: Gráficos 3D, calculos ciêntíficos rápidos

Double 8 bytes
- Aplicação: Cálculos ciêntificos precisos, finanças.

Char 1 byte
- Aplicação processamento de texto, armazenamento de caractere ASCII

Bool: True of False
- Aplicação: Flags, condições
Unisgned: Remove a representação de números negativos
Const: Tamanho fixo mutável
  - Aplicação: Parâmetros fixos
String: Um vetor de Char
Array/Vetor: Conjunto de elementos do mesmo tipo.
Ponteiro: Armazena o endereço de uma variável
Struct: Agrupa vários dados de tipos diferentes em uma mesma estrutura
Listas: Cada nó aponta para o próximo elemento da estrutura
Pilha: Empilha elementos e remove do topo
Árvores: Cada nó pode ter filhos, se for um dado menor vai para direita e menor para esquerda.
Grafos/Matriz: Representa Conexões (Arestas) entre pontos (vertices)
- Imagem de cubo (Arestas/vértices)
